# Student Grade Management System

A Python program that manages student grades.

## Features
- Add student names and scores
- Calculate class average
- Display highest and lowest scores
- Auto grade based on Monash grading system

## Grading System
- HD: 80 and above
- D: 70 - 79
- C: 60 - 69
- P: 50 - 59
- Fail: below 50

## How to Run
1. Open terminal
2. Run: python main.py
3. Enter student names and scores
4. Type 'done' to finish
